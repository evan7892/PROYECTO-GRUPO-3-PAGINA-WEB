FORMULA DE BEER-LAMBERT PARA LA ABSORBANCIA

El código en Python define una función llamada "CalcularAbsorbancia" que aplica la ley de Beer-Lambert para calcular la absorbancia a partir del coeficiente de absorción molar, la longitud del camino óptico, la intensidad incidente y la intensidad transmitida ingresados por el usuario. Luego, se solicitan estos valores al usuario como números decimales. La función realiza el cálculo de la absorbancia utilizando la fórmula correspondiente y devuelve el resultado, que se almacena en la variable "absorbancia". Finalmente, el programa muestra el valor calculado de la absorbancia en la pantalla. Este código es útil en espectrofotometría para determinar la concentración de una sustancia en una solución a partir de la cantidad de luz absorbida.
